📖 火山云（Cloudflare Worker）部署教程 展开 ▾
⚠️ 为什么需要这个？
浏览器有 CORS 安全限制，HTML 页面无法直接调用火山引擎 API。GitHub Pages 是纯静态托管，不能运行后端。所以需要部署一个免费的 Cloudflare Worker 作为云端代理。
🎯 方案特点
• 完全免费（每天 10 万次请求）
• API Key 存在 Worker 环境变量里，不暴露给前端
• 全球 CDN 加速
• 一次部署，GitHub Pages 永久可用
• 简单直接，有 API Key 就能用，无需登录
📋 部署步骤（3分钟）
1. 注册 Cloudflare 账号（免费）：https://dash.cloudflare.com/sign-up
2. 进入 Workers & Pages → Create application → Create Worker
3. 命名（如 ark-proxy），点击 Deploy
4. 点击 Edit code，粘贴下方代理代码，Save and deploy
5. Settings → Variables → 添加环境变量：
　　变量名：ARK_API_KEY
　　值：你的火山引擎 API Key（如 ark-xxxxxxxx）
　　类型：选 Secret（加密）
6. Save and Deploy，获取 Worker 网址（如 https://ark-proxy.xxx.workers.dev）
📝 代理代码（复制粘贴到 Worker 编辑器） 
// 火山引擎 ARK API 代理（Agent Token 套餐版，纯 API Key 验证）
const ARK_BASE = 'https://ark.cn-beijing.volces.com/api/plan/v3';

const CORS_HEADERS = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Methods': 'GET, POST, OPTIONS',
  'Access-Control-Allow-Headers': 'Content-Type, Authorization',
  'Access-Control-Max-Age': '86400',
};

export default {
  async fetch(request, env) {
    if (request.method === 'OPTIONS') {
      return new Response(null, { headers: CORS_HEADERS });
    }
    if (request.method !== 'POST') {
      return new Response(JSON.stringify({error:'Only POST is supported'}), {
        status: 405, headers: {'Content-Type':'application/json', ...CORS_HEADERS}
      });
    }
    try {
      const body = await request.json();
      const authHeader = request.headers.get('Authorization');
      let apiKey = '';
      if (env.ARK_API_KEY) {
        apiKey = env.ARK_API_KEY;
      } else if (authHeader && authHeader.startsWith('Bearer ')) {
        apiKey = authHeader.slice(7);
      } else {
        return new Response(JSON.stringify({error:'未提供 API Key'}), {
          status: 401, headers: {'Content-Type':'application/json', ...CORS_HEADERS}
        });
      }
      const arkResponse = await fetch(ARK_BASE + '/chat/completions', {
        method: 'POST',
        headers: {'Content-Type':'application/json','Authorization':'Bearer '+apiKey},
        body: JSON.stringify(body),
      });
      if (body.stream && arkResponse.body) {
        return new Response(arkResponse.body, {
          status: arkResponse.status,
          headers: {'Content-Type':'text/event-stream','Cache-Control':'no-cache','Connection':'keep-alive', ...CORS_HEADERS}
        });
      }
      const responseData = await arkResponse.text();
      return new Response(responseData, {
        status: arkResponse.status,
        headers: {'Content-Type':'application/json', ...CORS_HEADERS}
      });
    } catch (error) {
      return new Response(JSON.stringify({error:'代理请求失败: '+error.message}), {
        status: 500, headers: {'Content-Type':'application/json', ...CORS_HEADERS}
      });
    }
  },
};
⚙️ 本工具配置说明
部署完成后，在本配置面板中：
• Base URL：填你的 Worker 网址（如 https://ark-proxy.xxx.workers.dev）
• API Key：留空即可（Key 已存在 Worker 里），也可填上
• 模型：默认 ark-code-latest（Auto 模式，自动选模型）
🔑 火山引擎 Key 获取
• 购买 Agent Token 套餐后获得，格式：ark-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx-xxxxxx
• 火山引擎控制台 → 右上角头像 → API 访问密钥
• Base URL 必须用 /api/plan/v3（套餐专用），不是 /api/v3
✅ 验证部署
填好 Base URL 后点击"测试连接"，显示"已连接 · 火山云"即成功。有 API Key 就能用，无需登录。
