样式1: 极简现代

极简主义强调去繁就简，通过清晰的排版和充足的留白创造舒适的阅读体验。这种风格适合各类正式文章，让内容成为焦点，减少视觉干扰。 
.paragraph-style-1 {
  font-family: 'Inter', sans-serif;
  color: #374151;
  line-height: 1.7;
  font-size: 1rem;
  margin-bottom: 1.5rem;
  text-align: left;
}
